Instruções para a compilação e execução
=======================================
1. Para compilar  jogo, deve  entrar no terminal. Dentro do diretorio,dá o comando *make*.
>~/../tp1cg$ `make`
2. Depois da compilação, roda o arquivo executávell que foi criado fazendo 
>~/../tp1cg$ `./tp1compgraf`
